Installation
------------


1. Install from https://github.com/eoscanada/eosc/releases

**or**

2. Build from source with:

```bash
go get -u -v github.com/eoscanada/eosc/eosc
```

**or**

3. If you are on **MacOS** and use **Homebrew**:

```
brew install eoscanada/tap/eosc
```
